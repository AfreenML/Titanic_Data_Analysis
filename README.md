# Titanic_Data_Analysis

This project analyzes the Titanic dataset to understand survival patterns.

Key Insights:
- Females had a higher survival rate than males.
- First-class passengers were more likely to survive.
- Third-class had the highest number of passengers but the lowest survival rate.
- Younger passengers had better survival chances compared to older passengers.

Tools Used:
- Pandas
- Matplotlib
- Seaborn
